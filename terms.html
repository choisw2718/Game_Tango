핵심 요약

  • 정상적인 플레이와 기록 저장 범위에서 이용해 주세요.
  • 광고와 외부 사이트에는 각 제공자의 정책이 적용됩니다.

서비스 설명

Tango는 브라우저에서 플레이하는 논리 퍼즐 게임입니다. 사용자는 게스트 모드로 플레이하거나 닉네임 계정을 사용해 진행도를 저장할 수 있습니다.

사용자 책임

사용자는 퍼즐 플레이와 기록 저장 기능을 정상적인 방식으로 이용해야 합니다. 자동화된 요청, 서비스 방해, 타인의 닉네임 또는 연결 코드 오용, 부정한 기록 조작은 허용되지 않습니다.

콘텐츠와 지식재산

Tango의 화면 구성, 퍼즐 데이터, 코드, 설명 문구는 이 프로젝트의 운영 목적에 따라 제공됩니다. 별도 허가 없이 서비스를 사칭하거나 콘텐츠를 복제해 혼동을 일으키는 방식으로 사용할 수 없습니다.

광고와 외부 링크

사이트에는 Google AdSense 또는 기타 외부 서비스가 제공하는 광고와 링크가 표시될 수 있습니다. 외부 사이트의 내용, 결제, 개인정보 처리 방식은 해당 사이트의 정책을 따릅니다.

변경과 문의

운영상 필요한 경우 약관과 정책이 업데이트될 수 있습니다. 오류 신고, 정책 문의, 계정 관련 요청은 문의 페이지의 GitHub 이슈 링크를 사용해 주세요.
