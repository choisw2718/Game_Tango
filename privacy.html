핵심 요약
  • 게스트 진행도는 현재 브라우저에 저장됩니다.
  • 닉네임 계정 정보는 진행도 동기화에만 사용됩니다.
  • 광고가 게재되면 Google 등 제3자가 쿠키와 식별자를 사용할 수 있습니다.

수집하거나 저장하는 정보

게스트 모드에서는 진행도와 플레이 시간이 사용자의 브라우저 localStorage에만 저장됩니다. 닉네임 계정을 만들거나 연결하면 Supabase에 닉네임, 진행도, 스테이지 완료 기록, 기기 연결 코드와 같은 서비스 운영 정보가 저장될 수 있습니다.

정보 사용 목적

저장된 정보는 퍼즐 진행도 복원, 여러 기기 간 동기화, 스테이지 기록 표시, 오류 대응을 위해 사용됩니다. Tango는 사용자가 입력한 닉네임을 결제 정보나 민감 정보로 사용하지 않습니다.

광고 및 쿠키

이 사이트에 광고가 게재되는 경우 Google을 포함한 제3자 광고 제공업체는 사용자의 이 사이트 또는 다른 사이트 방문 기록을 기반으로 광고를 제공하기 위해 쿠키를 사용할 수 있습니다. Google의 광고 쿠키 사용은 Google과 파트너가 사용자에게 개인 맞춤 광고를 제공할 수 있게 합니다.

광고 게재 과정에서 제3자가 사용자의 브라우저에 쿠키를 저장하거나 읽을 수 있으며, 웹 비콘, IP 주소 또는 기타 식별자를 사용해 정보를 수집할 수 있습니다. Google이 파트너 사이트의 정보를 처리하는 방식은 Google의 파트너 사이트 데이터 사용 안내에서 확인할 수 있습니다.

Google 광고 설정 개인 맞춤 광고 관리 AboutAds 선택 페이지 제3자 맞춤 광고 선택 해제

외부 서비스

Tango는 Supabase를 계정 및 진행도 동기화에 사용하고, Supabase JavaScript 라이브러리를 jsDelivr CDN에서 불러옵니다. 각 외부 서비스는 자체 보안 및 개인정보 정책에 따라 데이터를 처리할 수 있습니다.

문의와 삭제 요청

개인정보 관련 요청, 닉네임 계정 삭제 요청, 오류 제보는 GitHub 이슈를 통해 문의할 수 있습니다. 공개 이슈에는 연결 코드, 개인 연락처, 기타 민감 정보를 포함하지 마세요.
