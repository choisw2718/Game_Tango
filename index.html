About Tango

Tango는 짧게 집중해서 풀 수 있는 논리 퍼즐입니다

각 스테이지는 검은돌과 흰돌을 배치해 행, 열, 인접 조건을 모두 만족시키는 문제입니다. 이 사이트는 브라우저에서 바로 플레이할 수 있도록 제작되었고, 공개된 퍼즐 데이터는 난이도와 크기별로 나뉘어 있습니다. 퍼즐 풀이 화면, 힌트, 체크포인트, 진행도 저장 기능은 모두 Tango 플레이 경험을 위해 직접 구현되었습니다.

Rules

기본 규칙

  • 각 행과 열에는 검은돌과 흰돌이 같은 개수로 들어갑니다.
  • 같은 돌이 가로 또는 세로로 3개 연속되면 안 됩니다.
  • = 표시는 양쪽 칸이 같은 돌이어야 한다는 뜻입니다.
  • x 표시는 양쪽 칸이 서로 다른 돌이어야 한다는 뜻입니다.
  • 빈칸을 누를 때마다 검은돌, 흰돌, 빈칸 순서로 바뀝니다.

Puzzle Library

스테이지와 난이도

Tango는 6 x 6과 8 x 8 보드를 제공하며, 쉬움과 어려움 난이도로 스테이지를 구분합니다. 각 문제는 고유한 조건과 시작 칸을 갖고, 브라우저에는 정답이 직접 노출되지 않도록 공개용 퍼즐 데이터만 포함합니다. 진행도는 게스트 모드에서는 이 브라우저에 저장되고, 닉네임 계정을 사용하면 여러 기기에서 이어서 플레이할 수 있습니다.

Privacy

개인정보와 광고 고지

Tango는 게스트 진행도를 브라우저 localStorage에 저장합니다. 닉네임 계정을 사용하는 경우 Supabase를 통해 닉네임, 진행도, 연결 코드, 스테이지 기록을 저장할 수 있습니다. Google을 포함한 제3자 광고 제공업체는 광고 제공을 위해 쿠키를 사용할 수 있으며, 사용자는 Google 광고 설정에서 개인 맞춤 광고를 관리할 수 있습니다.

게임 규칙

빈칸을 눌러 검은돌과 흰돌을 채우세요. 한 번 누르면 검은돌, 두 번 누르면 흰돌, 세 번 누르면 다시 빈칸이 됩니다.

  1. 각 행과 열에는 검은돌과 흰돌이 같은 개수로 들어갑니다.
  2. 같은 돌이 가로 또는 세로로 3개 연속되면 안 됩니다.
  3. = 표시는 양쪽 칸이 같은 돌이어야 합니다.
  4. x 표시는 양쪽 칸이 서로 다른 돌이어야 합니다.
  5. 모든 조건을 만족하면 Stage가 완료되고 다음 Stage가 열립니다.
