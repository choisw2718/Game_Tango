How to play

Tango 규칙과 기능

빈칸을 눌러 검은돌과 흰돌을 채우세요. 한 번 누르면 검은돌, 두 번 누르면 흰돌, 세 번 누르면 다시 빈칸이 됩니다.

  1. 각 행과 열에는 검은돌과 흰돌이 같은 개수로 들어갑니다.
  2. 같은 돌이 가로 또는 세로로 3개 연속되면 안 됩니다.
  3. = 표시는 양쪽 칸이 같은 돌이어야 합니다.
  4. x 표시는 양쪽 칸이 서로 다른 돌이어야 합니다.
  5. 모든 조건을 만족하면 Stage가 완료되고 다음 Stage가 열립니다.

사용할 수 있는 기능

  • 힌트: 막혔을 때 다음에 살펴볼 칸과 이유를 알려줍니다.
  • 체크포인트 저장: 현재 보드 상태를 저장해 둡니다.
  • 체크포인트 복원: 저장해 둔 보드 상태로 되돌립니다.
  • 보드 비우기: 처음 주어진 칸만 남기고 다시 시작합니다.
  • 다음 Stage: 문제를 완료하면 다음 Stage로 이동할 수 있습니다.
